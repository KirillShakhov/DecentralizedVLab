# СПИСОК СОКРАЩЕНИЙ И УСЛОВНЫХ ОБОЗНАЧЕНИЙ

API     – Application Programming Interface (программный интерфейс приложения)

CDN     – Content Delivery Network (сеть доставки содержимого)

CmRDT   – Operation-based Conflict-free Replicated Data Type

CRDT    – Conflict-free Replicated Data Type (бесконфликтный реплицируемый тип данных)

CvRDT   – State-based Conflict-free Replicated Data Type

DOM     – Document Object Model (объектная модель документа)

DoS     – Denial of Service (отказ в обслуживании)

ER      – Entity-Relationship (модель «сущность — связь»)

GC      – Garbage Collector (сборщик мусора)

HMR     – Hot Module Replacement (горячая замена модулей)

HTTPS   – HyperText Transfer Protocol Secure

IDB     – IndexedDB

IDE     – Integrated Development Environment (интегрированная среда разработки)

JIT     – Just-in-Time compilation (компиляция «на лету»)

JSON    – JavaScript Object Notation

LMS     – Learning Management System (система управления обучением)

LTI     – Learning Tools Interoperability (стандарт интеграции учебных инструментов)

LWW     – Last-Write-Wins (стратегия разрешения конфликтов)

MEMFS   – In-memory File System (файловая система Emscripten в памяти)

MOOC    – Massive Open Online Course (массовый открытый онлайн-курс)

OT      – Operational Transformation (операциональная трансформация)

P2P     – Peer-to-Peer (одноранговая сеть)

PWA     – Progressive Web Application (прогрессивное веб-приложение)

RAM     – Random Access Memory (оперативная память)

RPC     – Remote Procedure Call (удалённый вызов процедур)

RTT     – Round-Trip Time (время прохождения сетевого запроса туда и обратно)

SEC     – Strong Eventual Consistency (сильная конечная согласованность)

SIMD    – Single Instruction Multiple Data (одна инструкция — много данных)

SPA     – Single Page Application (одностраничное приложение)

SSE     – Server-Sent Events (события, отправляемые сервером)

UDC     – Universal Decimal Classification

UI      – User Interface (пользовательский интерфейс)

UML     – Unified Modeling Language (унифицированный язык моделирования)

UX      – User Experience (пользовательский опыт)

vCPU    – virtual CPU (виртуальное процессорное ядро)

VM      – Virtual Machine (виртуальная машина)

W3C     – World Wide Web Consortium

WASM    – WebAssembly

YATA    – Yet Another Transformation Approach (алгоритм CRDT для последовательностей)

ВКР     – выпускная квалификационная работа

ГИА     – государственная итоговая аттестация

ЛР      – лабораторная работа

ОП      – образовательная программа

ОС      – операционная система

ПО      – программное обеспечение

СУБД    – система управления базами данных

ТЗ      – техническое задание

ФТ      – функциональное требование

НФТ     – нефункциональное требование

ФС      – файловая система
