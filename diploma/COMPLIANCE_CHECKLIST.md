**Все Markdown-исходники готовы к конвертации.** Рекомендованная команда для конвертации в DOCX через Pandoc:

```bash
pandoc -s 00_contents.md 00_abbreviations.md 00_terms.md \
       01_introduction.md \
       02_chapter1.md 03_chapter2.md 04_chapter3.md 05_chapter4.md \
       06_conclusion.md 07_references.md 08_appendix.md \
       --reference-doc=itmo-template.docx \
       -o diploma.docx
```

(Файл `itmo-template.docx` — стилевой шаблон с полями, шрифтом, нумерацией страниц и стилями заголовков по требованиям ИТМО.)
